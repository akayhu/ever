# Tooltip
## Usage
```javascript
import Tooltip from 'src/client/component_common/tooltip';
<Tooltip
  icon = "help circle icon"
  width = "250px" 
  content="帳號設定的居住地完成後，需重新整理此頁面居住地才會更新。"
/>
```
## Props
- #### icon: [String] - Default:　'icon class name'
> option 
> ex: help circle icon 

- #### content: [String] 
> 塞入字串 

- #### content: [String] - Default: 250px 
> tooltip width
