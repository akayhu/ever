常見問題

104職涯社群常見問題：

1. 問: 104職涯社群是什麼？
答: 104職涯社群是104集團推出的職涯社群平台。在104職涯社群，您可以看到與自己同職類或您有興趣的職類會員們都在關注哪些專業資訊，並可與之互動，交換名片，拓展人脈；也可藉著由自己的熱情分享與專業見解，讓更多人能肯定您的專業，讓眾人看到您的智慧、經驗；更可以透過104累積多年的資料分析，進而吸引到對您職涯有幫助的貴人。現在開始在104職涯社群分享您的職涯心得或獨到、專業的見解吧！為自己贏到更多肯定，領先眾人讓貴人優先看到您的價值！
2. 問: 如何設定取消訂閱電子報?
答: 步驟一:請登入後，點入右上角小圖像旁倒三角形符號
步驟二:點選【管理電子報】，於此畫面中，您可設定電子報寄送頻率或選擇【取消訂閱】，最後點選【儲存】即可設定完成
3. 問: 如何發表文章
答: 步驟一:請點左上角104職涯社群LOGO回首頁，在選單下方有一個【發表】及鉛筆符號，點擊後出現發文視窗
步驟二:請輸入標題、分享內容、設定您要讓別人肯定的專業與特質，點【發佈】
步驟三:選擇文章分類主題（必選，最多三個），點【發佈】即完成
4. 問: 如何刪除文章
答: 若要刪除或編輯文章，請點入文章右上角鉛筆圖示，選擇刪除或編輯即可
5. 問: 如何肯定他人文章
答: 步驟一:點入您想肯定的文章，文末:「根據這篇分享，你想肯定對方什麼專業？」下方點選【+新增】
步驟二:自行輸入肯定內容。 提醒您，可在【+新增】輸入一兩個字是您想肯定對方的相關字(如:激勵、鼓勵…)，會出現相關字選單，選擇其一內容按”肯定”即可，無法直接自行輸入如:【很不錯】這些非表單設定的文字。
6. 問: 目前104職涯社群支援的瀏覽器版本?
答: 目前104職涯社群支援瀏覽器版本如下，如版本不足無法正常使用104職涯社群所有功能喔!
1. chrome : 版本26以上(含)
2. IE: 版本10以上(含)
3. Firefox火狐 : 版本20以上(含)
4. safari : 版本6以上(含)
7. 問: 忘記密碼如何登入?
答: 請依下列步驟重新設定您的新密碼:
步驟一:請在登入頁點「忘記密碼」，出現「重設您的密碼」，填寫曾驗證過的電子郵件信箱及驗證碼，點選「繼續」
步驟二:此時密碼重設信送至您的電子郵件信箱，您會收到一封內含修改密碼連結的信，點選連結即可設定您的新密碼 提醒您，如您遲遲未收到重設密碼信件，請檢查您的垃圾郵件夾
8. 問: 忘記帳號如何登入?
答: 如您忘記帳號，請依下列管道取得:
管道一:如您為my 104會員，您的身分證字號即為您的帳號
管道二:如您以Facebook加入會員者，請點選藍框，登入Facebook後即可使用
管道三:可在登入頁點「忘記帳號」，輸入任一可辨識您的資訊，亦可「+選取生日」提高準確率搜尋，選擇”這是我”會出現:「我知道帳號了，謝謝」點入這排字，回到登入頁登入即可
